Flappy Bird Game

The complete retro game in French!

C# Game
Development

June 2022

Flappy Bird is a simple game in which users need to hit the space bar to make the flappy bird fly and avoid incoming green pipes. At just over 60 lines, the code behind this game is simple having few variables for pipe speed, bird gravity and flight. Together with creative animations and fun fonts, the game is highly addictive as players can compete for the highest score!
